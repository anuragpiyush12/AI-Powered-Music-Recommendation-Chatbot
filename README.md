## Getting Started

### Prerequisites

This project requires the following dependencies:

* **Programming Language:** JupyterNotebook
* **Package Manager:** Pip

### Installation

Build AI-Powered-Music-Recommendation-Chatbot from the source and install dependencies:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/anuragpiyush12/AI-Powered-Music-Recommendation-Chatbot](https://github.com/anuragpiyush12/AI-Powered-Music-Recommendation-Chatbot)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd AI-Powered-Music-Recommendation-Chatbot
    ```

3.  **Install the dependencies:**

    Using `pip`:
    ```bash
    pip install -r requirements.txt
    ```
